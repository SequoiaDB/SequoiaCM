omadmin 提供 om server 配置管理功能

## 参数 ##

|参数      |缩写        |描述          |
|----------|-----------|--------------|
|--help    |-h         |获取帮助文档  |
|--version |-v         |获取版本信息  |

>  **Note:**
>
>  * 获取 omadmin 的帮助文档:omadmin.sh -h
>
>  * 获取特定子命令的帮助文档:omadmin.sh -h subcommand
