ftadmin 提供 fulltext server 配置管理功能

##参数##

|参数      |缩写        |描述          |
|----------|-----------|--------------|
|--help    |-h         |获取帮助文档  |
|--version |-v         |获取版本信息  |

>  **Note:**
>
>  * 获取 ftadmin 的帮助文档:ftadmin.sh -h
>
>  * 获取特定子命令的帮助文档:ftadmin.sh -h subcommand
