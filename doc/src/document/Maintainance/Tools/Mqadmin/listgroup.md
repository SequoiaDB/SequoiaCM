listgroup 子命令提供列取消费组的功能。

##子命令选项##

|选项       |缩写 |描述                                                   |是否必填|
|-----------|-----|----------------------------------------------------|--------|
|--name     |-n   |查看指定消费组                                             |否      |

###示例###

列取消费组

   ```lang-javascript
   $  mqadmin.sh listgroup
   ```
