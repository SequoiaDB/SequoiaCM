listtopic 子命令提供列取主题的功能。

##子命令选项##

|选项       |缩写 |描述                                                   |是否必填|
|-----------|-----|----------------------------------------------------|--------|
|--name     |-n   |查看指定主题                                             |否      |

###示例###

列取主题

   ```lang-javascript
   $  mqadmin.sh listtopic 
   ```
