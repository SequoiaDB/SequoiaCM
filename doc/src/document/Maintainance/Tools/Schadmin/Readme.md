schadmin 提供 schedule server 配置管理功能

##参数##

|参数      |缩写        |描述          |
|----------|-----------|--------------|
|--help    |-h         |获取帮助文档  |
|--version |-v         |获取版本信息  |

>  **Note:**
>
>  * 获取 schadmin 的帮助文档:schadmin.sh -h
>
>  * 获取特定子命令的帮助文档:schadmin.sh -h subcommand
