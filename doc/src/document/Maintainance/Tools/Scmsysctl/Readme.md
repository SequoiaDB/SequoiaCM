scmsysctl 工具提供所有服务节点管理相关的功能

##参数##
|参数      |缩写  |描述          |
|----------|------|--------------|
|--help    |-h    |获取帮助文档  |
|--version |-v    |获取版本信息  |
>  **Note:**
>
>  * 获取 scmsysctl 的帮助文档:scmsysctl.sh -h
>
>  * 获取特定子命令的帮助文档:scmsysctl.sh -h subcommand

##子命令列表##
|子命令       |描述            |
|-------------|----------------|
|[start][start]        |启动节点        |
|[stop][stop]         |停止节点        |
|[list][list]          |list节点        |
